## VSIX Extensions

VSIX is a Visual Studio extension installer (<https://marketplace.visualstudio.com/>).
Install by double-clicking the `.vsix` file after Visual Studio is installed.

- `EditorGuidelines.vsix`
- `NET_Portability_Analyzer_v2.7.0.vsix`
