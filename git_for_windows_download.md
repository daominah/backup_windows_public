## Git LFS

GitHub blocks files larger than 100 MiB.
To track files beyond this limit, use Git Large File Storage (Git LFS).

Example commands:

```sh
git lfs install

git lfs track --filename CALD3_installer_d201411.zip  # this creates file ".gitattributes"
```
