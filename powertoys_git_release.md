# PowerToys Download

<https://github.com/microsoft/PowerToys/releases>

## Useful utilities

PowerToys has a lot of utilities, but we only need:

### FancyZones

Dragging a window to a region resizes it to pre-defined sizes.

### Hosts File Editor

Convenient way to edit the hosts file. Also has a button to open the file directly.

### Keyboard Manager

Remap keys and shortcuts:

| New shortcut                           | Default                        | Meaning                                                                                                                                            |
|----------------------------------------|--------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| **Win** + **Shift** + **1**            | **Win** + **Shift** + **Left** | Move window to the left monitor                                                                                                                    |
| **Win** + **Ctrl** + **Shift** + **B** | **B**                          | Disable this combination. It restarts the graphics driver, which turns off the external monitor. Very annoying when we select all Barracks in AoE. |
| **Ctrl** + **`**                       | **Alt** + **Tab**              | Switch windows. Makes the shortcut work similarly to macOS behavior.                                                                               |
| **Win** + **D**                        | **Ctrl** + **D**               | Interrupt Terminal. Similarly to macOS behavior after remap `Win` key to their `control`                                                           |
| **Win** + **C**                        | **Ctrl** + **C**               | Interrupt Terminal.                                                                                                                                |
