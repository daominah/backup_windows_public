### Wireshark vs browser DevTools (e.g. Chrome)

| Capability                | Wireshark | Browser DevTools |
|---------------------------|-----------|------------------|
| See all apps traffic      | Yes       | No               |
| See browser traffic       | Yes       | Yes              |
| HTTP/HTTPS inspection     | Basic     | Excellent        |
| Low-level (TCP, DNS)      | Yes       | Limited          |
| Packet-level detail       | Yes       | No               |
| Easy to use               | No        | Yes              |
| Setup required            | Yes       | No               |
| Works outside browser     | Yes       | No               |
| Debug deep network issues | Yes       | Limited          |

### Simple takeaway

- DevTools: fast and easy for web
- Wireshark: powerful for deep network debugging
