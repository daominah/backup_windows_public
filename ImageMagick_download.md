### v6.9.13 (2026-03)

`ImageMagick-6.9.13-42-Q16-HDRI-x64-dll.exe`

Common CLI tools:

- `convert`: convert between formats, resize, crop, rotate, apply filters
- `identify`: display image metadata (format, dimensions, color depth, etc.)
- `mogrify`: same as `convert` but overwrites the original file in place (batch-friendly)
